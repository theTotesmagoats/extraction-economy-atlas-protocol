# Extraction Economy: ATLAS Protocol

A browser-based economic simulation game where players manage a tech company's extraction strategies while balancing growth, profitability, and ethical considerations.

## Overview

This is the source code for "EXTRACTION ECONOMY v26.2 - ATLAS Protocol" - a complex web application that models platform economics and the adversarial incentive structures driving modern markets.

## Features

- Dynamic economic simulation with user acquisition, retention, and monetization mechanics
- Multiple market segments (Hobbyists, Indie Pros, Studio Pros, Enterprise)
- Strategic protocol choices that affect your company's alignment on the Free Market to Extraction Monopoly spectrum
- Real-time UI with gauges tracking key metrics (R&D, Dependency, Satisfaction, Political Capital, etc.)
- Interactive inbox system with memos and decision points
- Annual shareholder reports and performance grading
- Exit strategy mechanics for IPO or PE sale

## Technical Details

- Single HTML file containing all CSS styling and JavaScript logic
- No external dependencies required - runs entirely in the browser
- Uses Web Audio API for sound effects
- LocalStorage for saving high scores
